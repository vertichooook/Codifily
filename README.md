Процесс обучения разбит на удобные шаги: от изучения теории и прохождения квизов до написания реального кода с мгновенной проверкой. \n\nНикаких скучных лекций. Только практика, современный дизайн и элементы геймификации!\n\n## Главные особенности\n\n- **Встроенная IDE (Skulpt)** — пишите и запускайте Python-код прямо в браузере без установки дополнительных программ.\n- **Система прогресса** — зарабатывайте XP, повышайте свой ранг и отслеживайте успехи в личном кабинете.\n- **Умный дизайн** — поддержка светлой и темной тем оформления (Glassmorphism), плавные анимации и кастомные элементы интерфейса.\n- **Мультиязычность** — интерфейс полностью переведен на русский, английский и немецкий языки.\n- **Интерактивный справочник** — быстрый и удобный поиск по теории с современной визуализацией.\n- **Геймификация** — визуальный и звуковой отклик при правильном решении задач для повышения вовлеченности.\n\n## Технологии\n\n- **Backend**: Flask (Python)\n- **Frontend**: HTML, CSS (Bootstrap 5), JavaScript\n- **База данных**: SQLite\n- **Исполнение кода**: Skulpt (выполнение Python в браузере)\n\n## Установка и запуск\n\n1. Клонируйте репозиторий:\n   ```bash\n   git clone <url-репозитория>\n   ```\n\n2. Перейдите в директорию проекта:\n   ```bash\n   cd LearningApp\n   ```\n\n3. Создайте и активируйте виртуальное окружение:\n   ```bash\n   python -m venv .venv\n   # Для Windows:\n   .venv\\Scripts\\activate\n   # Для Linux/macOS:\n   source .venv/bin/activate\n   ```\n\n4. Установите зависимости (если есть requirements.txt):\n   ```bash\n   pip install -r requirements.txt\n   ```\n\n5. Инициализируйте или запустите приложение:\n   ```bash\n   python app.py\n   ```\n\n6. Откройте браузер и перейдите по адресу:\n   ```text\n   http://127.0.0.1:5000\n   ```\n\n## Структура проекта\n\n- `app.py` — основной файл приложения Flask.\n- `translate_course.py` — скрипт для генерации переводов курсов на другие языки.\n- `static/` — статические файлы (такие как CSS, JavaScript, изображения и `courses.json`).\n- `templates/` — HTML-шаблоны страниц.\n- `instance/` — папка с файлом базы данных SQLite.
