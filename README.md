Привет! Это Codifi.ly — небольшой образовательный проект, который позволяет изучать Python проще и интереснее.

Идея в том, чтобы не просто читать скучную теорию в учебнике, а сразу пробовать писать код, не уходя со страницы и не устанавливая никаких программ на свой компьютер. Я добавил сюда встроенный редактор кода, тесты, внутренний справочник и систему достижений, чтобы процесс был больше похож на игру, чем на лекцию.

## Что внутри?

Вот некоторые вещи, которые реализованы в проекте:

- **Встроенная песочница.** Можно писать Python-скрипты, и они будут выполняться прямо в браузере с помощью Skulpt. Редактор сам подсветит код и сразу покажет результат работы программы.
- **Немного геймификации.** За правильные ответы начисляется опыт (XP) и растет ваш ранг, а при успешном решении задачек на экране появляется праздничное конфетти. Мелочь, а приятно!
- **Внешний вид.** Я постарался сделать так, чтобы платформой было удобно и приятно пользоваться. Есть светлая и тёмная темы, плавные переходы между страницами и элементы в стиле эффекта матового стекла (Glassmorphism).
- **Поддержка нескольких языков.** Интерфейс и сами уроки изначально переведены на русский, английский и немного на немецкий.
- **Удобный справочник.** Если забыли какую-то конструкцию языка, можно быстро найти нужную информацию в местной базе знаний без отрыва от задачи.

## На чем это всё работает

Вся техническая часть устроена довольно просто:
- Сервер написан на Flask (с использованием Python).
- В качестве базы данных используется SQLite — для локального проекта этого вполне достаточно.
- Фронтенд собран из HTML, JavaScript и компонентов Bootstrap 5.
- За исполнение кода пользователя целиком отвечает библиотека Skulpt (работает на стороне клиента).

## Если хотите запустить у себя

Чтобы пощупать проект локально, понадобится сделать несколько шагов:

1. Скачайте исходный код к себе:
   ```bash
   git clone <ссылка-на-репозиторий>
   cd LearningApp
   ```

2. Рекомендую создать виртуальное окружение, чтобы ставить зависимости было безопасно:
   ```bash
   python -m venv .venv
   ```

3. Активируйте окружение.
   Если у вас Windows:
   ```bash
   .venv\Scripts\activate
   ```
   А если Mac или Linux:
   ```bash
   source .venv/bin/activate
   ```

4. Установите нужные библиотеки:
   ```bash
   pip install -r requirements.txt
   ```

5. И запустите сервер разработки:
   ```bash
   python app.py
   ```

Затем просто откройте браузер и перейдите на страницу `http://127.0.0.1:5000`.

## Что где лежит

Для тех, кто захочет покопаться в коде, вот краткий экскурс:
- В `app.py` находится вся серверная логика и маршруты приложения.
- В папке `templates` лежат HTML-шаблоны страниц, а в папке `static` — стили, скрипты и наши загруженные курсы (в файлике `courses.json`).
- `translate_course.py` — это просто полезный скрипт, через который я прогонял тексты уроков для их автоперевода на другие языки.
- `instance` — здесь платформа хранит свой файл с базой SQLite.
